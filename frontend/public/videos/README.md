# Video Assets

Place your video files here:

- `hero-video.mp4` - Main hero video (recommended: 1920x1080, 30fps, H.264)
- `hero-video.webm` - WebM version for better browser support

## Video Specifications:
- Resolution: 1920x1080 or higher
- Duration: 10-30 seconds (looping)
- Format: MP4 (H.264) and WebM
- Size: Keep under 10MB for web optimization
- Content: Professional event management related footage

## Fallback Image:
- `hero-fallback.jpg` - Static image fallback (1920x1080)
